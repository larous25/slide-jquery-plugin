jQuery simple slide for  ul element
===================================

#use:
1. create a div
2. crate a ul 
3. use `slide()` with a div element itself contains an unordered list

#options
the options is the object with next attributes
indexElement: boolean 
arrow : boolean
textArrows : object
example 
`{`
`indexElement:true,`
` arrows:true,`
` textArrows:{`
`    right:'<',`
`     left:'>'}`
`}`
